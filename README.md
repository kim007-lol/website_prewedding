# Ayam & Bebek Wedding

This is a responsive web application for a wedding invitation, designed to provide guests with event details, the couple's story, an RSVP form, and gift information.

## Features

- **Responsive Design**: Built using Bootstrap 5 to ensure the website looks great on all devices (mobile, tablet, desktop).
- **Countdown Timer**: A customizable countdown timer displaying the time remaining until the wedding day.
- **Off-canvas Sidebar Navigation**: A modern, mobile-friendly navigation sidebar for easy access to different sections.
- **Our Story Timeline**: An interactive timeline showcasing key moments in the couple's relationship.
- **Photo Gallery**: A built-in modal lightbox gallery displaying the couple's photos.
- **RSVP Form**: A functional RSVP form to collect guest confirmation details.
- **Digital Gifts**: Information for guests who wish to provide digital gifts via bank transfer.
- **Background Audio**: Integrated background music that auto-plays with a toggle control button.

## Technologies Used

- **HTML5**: For the structure of the web pages.
- **CSS3 (Custom & Bootstrap 5)**: For styling, layout, and responsive utilities.
- **JavaScript (Vanilla & Bootstrap 5 Bundle)**: For interactive elements such as the countdown timer, audio controls, and lightbox gallery.
- **simplyCountdown.js**: A lightweight JavaScript library used to create the countdown timer.
- **Bootstrap Icons**: For integrating easily customizable SVGs.
- **bs5-lightbox**: For the built-in, responsive image gallery modal.

## How to Get Started

1.  Clone the repository to your local machine:
    `git clone https://github.com/kim007-lol/website_prewedding.git`
2.  Open the `index.html` file in your preferred web browser to view the invitation. No build process is required.

## Customization

The details in this template can be easily adjusted to suit your needs:

- **Audio File**: You can replace the audio file located at `audio/komang.mp3` with your preferred track.
- **Images**: Update the `bride.png`, `groom.png`, and other files placed inside the `img` and `img/gallery/thumbnail` folders. (Note: Ensure any image additions are not ignored by the `.gitignore`).
- **Dates & Text**: Edit `index.html` to change names, event details, story content, and the countdown timer parameters.
- **Theme Colors**: Modify CSS variables in `style.css` (e.g., `--red`, `--bg`) or add new classes to adjust the appearance.
